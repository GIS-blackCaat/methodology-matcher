## 📋 概述

任务方法论匹配器 - 为复杂长期任务自动匹配适配的方法论框架。

**核心理念**：
- **简单任务** → 直接执行
- **复杂任务** → 方法论匹配 → 配置 → 执行

---

## 🎯 功能特性

- ✅ 任务复杂度自动检测（简单/中等/复杂）
- ✅ 现代方法论库（费曼技巧、PDCA、OKR、Agile、设计思维、GTD）
- ✅ 中国古代方法论库（格物致知、知行合一、循序渐进、温故知新、学以致用）
- ✅ 复杂任务自动匹配方法论
- ✅ 简单任务直接执行

---

## 📊 任务分类

| 类型 | 时间跨度 | 步骤数量 | 执行方式 |
|------|---------|---------|---------|
| **简单** | <1 天 | ≤2 步 | 直接执行 ✅ |
| **中等** | 1-7 天 | 3-5 步 | 轻量方法论 ⚠️ |
| **复杂** | >7 天 | >5 步 | 完整方法论 🔴 |

---

## 📚 方法论库

### 现代方法论

- **费曼学习法** - 学习新知识、准备考试
- **PDCA 循环** - 持续改进、质量管理
- **OKR** - 目标管理、长期规划
- **Agile/Scrum** - 项目开发、产品迭代
- **设计思维** - 产品创新、问题解决
- **GTD** - 任务管理、时间管理

### 中国古代方法论

- **格物致知** - 研究学习、知识探索
- **知行合一** - 实践应用、技能转化
- **循序渐进** - 系统学习、知识构建
- **温故知新** - 记忆巩固、知识创新
- **学以致用** - 技能培养、能力提升

## 🚀 使用示例

### 复杂任务

```
任务：我想系统学习计算机 408 考研知识

分析：
- 时间跨度：6 个月 → 复杂 🔴
- 步骤数量：>10 步 → 复杂 🔴
- 需要系统规划：是 → 复杂 🔴

推荐方法论：
1. 费曼技巧（核心学习方法）
2. 循序渐进（知识构建顺序）
3. 温故知新（定期回顾）
```

### 简单任务

```
任务：北京今天天气怎么样？

分析：
- 时间跨度：<1 天 → 简单 ✅
- 步骤数量：1 步 → 简单 ✅
- 无需系统规划 → 简单 ✅

执行方式：直接调用 weather skill ✅
```
